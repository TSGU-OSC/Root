## 其他
### 安装教程
1. 下载代码
2. npm run serve
3. 浏览器访问http://localhost:3000/Root


### 提交代码


首先fork

然后进行修改或新建代码

同步代码到自己的仓库

新建pull Request，需标明提交的内容

最后等待管理员审批

### 新手注意
1. 文件名不要用特殊字符
2. 工程代码必须新建一个文件夹，禁止零散污染目录！
